# Word报告
## 简介
使用Word作为模板，关键地方使用占位符，用动态内容代替输出报告
## 使用方法

见测试代码

```
    <dependency>
	    <groupId>com.github.bobdeng</groupId>
	    <artifactId>wordreport</artifactId>
	    <version>Tag</version>
	</dependency>
	
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```
